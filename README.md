# llamanet gradio demo

## Install

### 1. Activate venv

First activate venv.

On Windows:

```
python -m venv env
env/Scripts/activate
```


On Linux/Mac:

```
python -m venv env
source env/bin/activate
```

### 2. Install dependencies

```
pip install -r requirements.txt
```

## Run

```
python app.py
```
